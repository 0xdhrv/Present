Inheritance

Inheritance is a way of creating a new class for using details of an existing class without modifying it.

The newly formed class is a derived class (or child class). Similarly, the existing class is a base class (or parent class). 


						# Parent Class

						class Parent:
							def __init__(self):
								self.parent_variable = "Parent Variable"
								print("Parent Object Created")
							
							def ParentMethod(self):
								return f"This is a Parent Method"
							
					

						# Child Class

						class Child(Parent):
							def __init__(self):
								super().__init__()
								self.child_variable = "Child Variable"
								print("Child Object Created")

							def ParentMethod(self):
								print("Inherited Method of Parent")
							
							def ChildMethod(self):
								print("This is a Child Method")
let's implement 

						>>> P = Parent()
						'Parent Object Created'
						>>> P.parent_variable
						'Parent Variable'
						>>> P.ParentMethod()
						'This is a Parent Method'
					

						>>> C = Child()
						'Child Object Created'
						>>> C.child_variable
						'Child Variable'
						>>> C.ChildMethod()
						'This is a Child Method'
						>>> C.ParentMethod()
						'Inherited Method of Parent'
Encapsulation

Encapsulation is a way to restrict access to methods and variables.

This prevents data from direct modification. 


						class Product:

						def __init__(self):
							self.__price = 1000
					
						def Sell(self):
							print("Selling Price: {}".format(self.__price))
					
						def setPrice(self, newprice):
							self.__price = newprice
					

						>>> Diamond = Product()
						>>> Diamond.Sell()
						'Selling Price : 1000'
						>>> Diamond.setPrice(2000)
						>>> Diamond.Sell()
						'Selling Price : 2000'
Polymorphism

Polymorphism is a way to use a common interface for multiple form (poly morph) of data.

The concept of polymorphism is to use the same method or operator for different kind of data. 


                        class Cat:
                            def Meow():
                                print("Cat can Meow")
                            def Fly():
                                print("Cat cannot Fly")
                        
                        class Crow:
                            def Meow():
                                print("Crow cannot Meow")
                            def Fly():
                                print("Crow can Fly")
                        
                        def flyingTest(bird):
                            bird.Fly()
                    

                        >>> Tom = Cat()
                        >>> Scare = Crow()
                        >>> flyTest(Tom)
                        'Cat cannot Fly'
                        >>> flyTest(Scare)
                        'Crow can Fly'
Abstraction

Abstraction is way of hiding the complexity and only showing the essential features of the object.

It means hiding the real implementation of a object.
